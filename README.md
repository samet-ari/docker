# Projet Docker - La Plateforme

## Description
Apprentissage de Docker : conteneurs, volumes, images et orchestration avec Docker Compose.

## Structure du projet
```
docker/
├── job01/              → Installation de Docker sur Debian
├── job02/              → Test avec hello-world et commandes de base
├── job03/              → Dockerfile et conteneur hello-world
├── job04/              → Conteneur SSH avec Dockerfile
├── job05/              → Alias Docker dans ~/.bashrc
├── job06/              → Volumes partagés entre conteneurs
├── job07/              → Docker Compose : nginx + FTP
├── job08/              → Conteneur nginx depuis Dockerfile
├── job09/              → Registry local + interface web
├── job10/              → Scripts bash install/clean Docker
├── job11/              → Portainer
└── pour_aller_plus_loin/ → Équivalent XAMPP avec Docker Compose
```

## Compétences acquises
- Créer et gérer des conteneurs Docker
- Écrire des Dockerfiles
- Utiliser Docker Compose
- Gérer des volumes et des réseaux
- Déployer un registry local
- Utiliser Portainer comme interface graphique

## Technologies utilisées
- Docker
- Debian
- nginx
- PHP
- MariaDB
- Portainer
